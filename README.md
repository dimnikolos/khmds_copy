# Εξαγωγή αναγνωριστικών ΚΗΜΔΣ

Ένα Python πρόγραμμα για την αυτόματη εξαγωγή κωδικών ΚΗΜΔΣ (ΑΔΑΜ-REQ/PROC/AWRD/SYMV/PAY) από τα PDF.

Το script εντοπίζει κωδικούς μορφής `..REQ...`, `..PROC...`, `..AWRD...`, `..SYMV...`, `..PAY...`, που όπως γνωρίζουμε δεν μπορούμε να το κάνουμε αντιγραφή από το PDF και το σημειώνουμε σε χαρτί για να το ξαναπληκτρολογήσουμε όπου χρειάζεται.

## Δυνατότητες

- **Έξυπνη αναζήτηση:** Διαβάζει απευθείας τα metadata για την ελαχιστοποίηση λαθών.
- **Αναζήτηση με regular expreissions:** Αναγνωρίζει αυτόματα τους τύπους: `REQ`, `PROC`, `AWRD` , `SYMV`, `PAY`.
- **Άμεση αντιγραφή και επικόλληση:** Αντιγράφει αυτόματα τον κωδικό που βρέθηκε στο πρόχειρο.

## 🛠️ Απαιτήσεις

- Python 3.x
- Βιβλιοθήκες: `pypdf`, `pyperclip`

## 📦 Εγκατάσταση

1.  Κλωνοποιήστε το repository ή κατεβάστε το script.
2.  Εγκαταστήστε τα dependencies:

```bash
pip install pypdf pyperclip
```
