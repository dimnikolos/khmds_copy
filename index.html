Αντιγραφή ΑΔΑΜ

Για να πάρετε τον κωδικό ΑΔΑΜ από το pdf το σημειώνετε πρώτα στο χαρτί και μετα τον πληκτρολογείτε;

Όχι πια!

Αντιγραφή του κωδικού ΑΔΑΜ (πρωτογενή, προσκλήσεις, αναθέσεις, συμβάσεις και εντάλματα) με ένα δεξί κλικ.

📥 Λήψη για Windows Προβολή στο GitHub
Εύκολη εγκατάσταση
  • Κατεβάστε το αρχείο εκτελέστε το.
Πώς λειτουργεί;
  • Ακρίβεια Raw Stream: Διαβάζει απευθείας τον κώδικα του PDF. Βρίσκει το ID ακόμα κι αν το κείμενο φαίνεται "κινέζικα" (mojibake).
  • Ενσωμάτωση στο Context Menu: Απλά κάντε Δεξί Κλικ → Αντιγραφή ΑΔΑΜ πάνω στο αρχείο.
  • Native Ειδοποιήσεις: Εμφανίζει επιβεβαίωση (Toast Notification) και αντιγράφει αυτόματα στο πρόχειρο.
  • Χωρίς Διαδικασίες IT: Εγκατάσταση χωρίς δικαιώματα διαχειριστή (No-Admin needed).

Ο κώδικας είναι ανοικτός και διαθέσιμος στο GitHub Repository.
